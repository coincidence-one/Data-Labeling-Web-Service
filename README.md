본 서비스는 한국공학대학교 졸업작품 AI 학습용 데이터 레이블링 시스템입니다.

# 개발환경
* 웹 어플리케이션
  * 사용언어: 타입스크립트 
  * 웹 프레임워크: node.js + express.js + react.js

* 사용자 맞춤 AI모델 구현
  * 사용언어: 파이썬
  * 라이브러리: Pytorch, YOLOv5 통한 객체 인식 AI 구현
 
* 데이터베이스
  * MongoDB - JSON 확장자 형식 같은 document 타입의 데이터 저장

* 호스팅 환경
  * AWS
